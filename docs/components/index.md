# Components

VUIII provides a comprehensive set of Vue 3 components for building modern web applications.

## Form Inputs

| Component | Description |
|-----------|-------------|
| [Input](/components/input) | Text input with icon support and validation states |
| [Textarea](/components/textarea) | Multi-line text input |
| [Select](/components/select) | Native select dropdown with flexible option formats |
| [Autocomplete](/components/autocomplete) | Searchable dropdown with keyboard navigation |
| [Checkbox](/components/checkbox) | Checkbox with toggle/switch variant |
| [CheckboxGroup](/components/checkbox-group) | Multi-select checkbox group |
| [RadioGroup](/components/radio-group) | Single-select radio button group |
| [RadioButtonGroup](/components/radio-button-group) | Button-styled radio group |
| [FilePicker](/components/file-picker) | File picker with drag-and-drop support |

## Buttons

| Component | Description |
|-----------|-------------|
| [Button](/components/button) | Polymorphic button with variants and icons |
| [IconButton](/components/icon-button) | Icon-only button |
| [ButtonGroup](/components/button-group) | Group of related buttons |
